## Admin Reset Password Edge Function

This function allows an authenticated **admin** to reset another user's password.

### Deploy

1. Login to Supabase CLI:
   `supabase login`
2. Link project (if not linked):
   `supabase link --project-ref <your-project-ref>`
3. Deploy function:
   `supabase functions deploy admin-reset-password`

### Frontend Env

Set this in your frontend `.env`:

`VITE_ADMIN_RESET_PASSWORD_ENDPOINT=https://<your-project-ref>.supabase.co/functions/v1/admin-reset-password`

### Optional Audit Table

If you want to store reset logs, run:

```sql
create table if not exists public.admin_password_resets (
  id bigint generated by default as identity primary key,
  admin_id uuid not null,
  target_user_id uuid not null,
  target_email text,
  created_at timestamptz not null default now()
);
```

If this table does not exist, password reset still works.
