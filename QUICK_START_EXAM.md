# Quick Start - Add Exam Questions ⚡

## Admin: Add Questions in 5 Steps

1. **Go to Courses** page
2. **Hover over course card** → 3 buttons appear
3. **Click purple button** (FileText icon) → Opens exam manager
4. **Fill the form:**
   - Question: Type your question
   - Options: Fill A, B, C, D
   - Click the correct letter (turns green)
   - Click "Add Question"
5. **Close and done!**

## Example

```
Question: "What is 2 + 2?"

Fill in:
[ ] Option A text: 3
[ ] Option B text: 4
[ ] Option C text: 5
[ ] Option D text: 6

Click: B ← Correct answer
```

## Student: Take Exam in 3 Steps

1. **Enroll** → Click "Enroll Now" on course
2. **Watch & Learn** → Video, notes, overview tabs
3. **Take Exam** → Click "Take Exam" tab → Answer questions

## System Handles

- ✅ Automatically grades exam
- ✅ Requires 70% to pass
- ✅ Generates certificate if passed
- ✅ Allows retry after 7 days if failed

That's it! 🎓
